Les 3

In de derde les kregen we een snelcursus website maken. Waarna we via oefeningen het allemaal beter leerden kennen en zo zelf konden beginnen aan het maken van onze eigen website. In HTML komt de code voor de vorm en inhoud van de website. In de CSS code komt de opmaak van alle elementen zoals tekst en kaders op de website. De niet-geziene code die we implementeerden in onze website vonden we op het internet. Aangezien deze code nog vrij basic is, is deze makkelijk te begrijpen.
